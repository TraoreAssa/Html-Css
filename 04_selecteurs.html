Les balises HTML

Les secteurs de type ciblent le éléments de la page HTML par les balises. Par exemple si nous voulons cibler tous nos titres de niveau <h4> de la page HTML nous utiliserons le secteur de type h4.

HTML <h4/> Bonjour à tous
<h4/> Au revoir à tous

CSS :
<h4>{
...}

Les sélecteurs de class/strong> permettent de cibler permettent de cibler un élément en fonction de la valeur de l"atribut class attaché à cet élément. Les sélecteurs de class permettent davantage de précision car ils permettent de cibler les éléments d'un groupe ayant une class identique plutôt que toutes les balises HTML de la page. Les sélecteursde class permettent d'appliquer la même mise en forme en attribuant la même valeur dans l'attribut class à différentes balises HTML. En CSS les class sont appelées précédes d'un point. les sélecteurs d'ID sont encore plus précis que les selecteurs de class, puisqu'ils ciblent un élément unique. Tout comme les sélecteurs de class utilisent la valeur d'une class présente dans le HTML, les sélecteurs d'ID utilisent la valeur de l'attribut ID.

/!/ATTENTION! Très important: peu importe la balise dans laquelle ils sont insérés, la valeur des attributs ID na peuvent être utilisés qu'une seule fois dans le fichier HTML. c'est pourquoi les ID sont réservés pour les éléments significatifs.
En CSS les ID sont appelés précédes d'un dièse (ou hash tag). Ici les sélecteurs d'ID ciblera uniquement l'élément HTML qui possède un atribut ID dont la valeur est "zone3"
