Aide Mémoire des sélecteurs CSS déja vue

  • Sélectionne Tous les éléments de la page html
  • .ma-joli-nav sélectionne tous les éléments ayant le nom de la class "ma ma-joli-nav"
  • #mon-unique-id sélectionne tous les éléments ayant le nom de mon id "mon-unique-id"
  • div, article, aside sélectionne toutes les balises <div>, <article> <aside>
  • nav a sélectionne les liens <a> inclut/imbriqués dans la balise <nav>
  • nav a hover sélectionne tous les liens de la nav et active une animation lors du survol de la souris

Propriétés CSS utilisées dans cette page

Décrivez leur usage et leurs valeur possibles

  1. Background: couleur du fond de l'élément
  2. Color: couleur de la police
  3. Float: effet flottant
  4. Font-size:Taille de la police
  5. Height: hauteur de l'élément
  6. Left: position à partir de la gauche de la fenêtre
  7. Line-height: hauteur de l'interligne
  8. Margin: marges exterieures
  9. Padding: marges interieures
  10. Position: position de l'élément dans ma page ( on n'a utilisé que la valeur (fixed) pour l'instant
  11. text-align: alignement du texte
  12. text-decoration: changement de valeurs héritées (ici le souligné est standard pour les liens sous tous les navigateurs est est supprimé
  13. Top: position à partir du haut de la fenêtre
  14. Transition: temps de transition lors d'une animation (ici sur les liens de la barre de menu)
  15. Width: largeur de l'élément par rapport à la taille de la fenêtre si on indique une valeur relative.
